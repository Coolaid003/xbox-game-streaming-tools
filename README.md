## Welcome!

Xbox Game Streaming Tools provide schemas and samples to enable game developers that are optimizing their games for Xbox game streaming.

To get started with Xbox game streaming you apply to the ID@Xbox program at http://www.xbox.com/en-us/Developers/id.

To learn more about optimizing your Xbox game for game streaming see the [developer documentation](https://docs.microsoft.com/en-us/gaming/game-streaming/).

## What's included

- [Schema](./touch-adaptation-kit/schemas) representating the capabilities of the [Touch Adaptation Kit](https://docs.microsoft.com/en-us/gaming/game-streaming/ux/touch-adaptation-kit/) that allows for your creation of custom touch layouts for your games.
- [Sample touch adaptation layouts](./touch-adaptation-kit/touch-adaptation-bundles) to get started.
- [Scripts](./touch-adaptation-kit/scripts) to assist in the development of touch daptation layouts.
